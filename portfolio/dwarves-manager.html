Dwarves Manager

Android game with more than 100,000 downloads.

Dwarves Manager was an Android management simulation game released in 2013, inspired by sandbox management games but adapted to a mobile experience. It started as a student project at ENIB, but I took it beyond and released the full game on the Play Store, where it reached more than 100,000 downloads. It was a small project but it was a great learning experience and a lot of fun to work sleepless nights on.

The game is actually very rough as of today standard, it has many issues and does not work well on modern Android devices (screen resolution issues). It is not available on Play Store anymore, due to API changes in Android. A sequel is currently in development, building on the same concept with a broader scope and more polished execution.

The game also featured original graphics and design from myself, and original music composed by a friend.
